Theorem 3.5.19. Row operations and determinant.

Let \(A\) be an \(n\times n\) matrix. Using the notation from DefinitionÂ 3.4.1 we have:

  1. \(\displaystyle \det(\underset{c\cdot r_i}{E}\, A)=c\det(A)\)

  2. \(\displaystyle \det(\underset{r_i\leftrightarrow r_j}{E}\, A)=-\det(A)\)

  3. \(\det(\underset{r_i+cr_j}{E}\, A)=\det(A)\text{.}\)

In particular, taking \(A=I\text{,}\) we have

\begin{align*} \det\underset{c\cdot r_i}{E}\amp =c \amp \det\underset{r_i\leftrightarrow r_j}{E}\amp =-1 \amp \det \underset{r_i+cr_j}{E}\amp =1 \text{.} \end{align*}
