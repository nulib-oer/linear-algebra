Item 4.
Given a subset \(S=\{\boldv_1,\boldv_2,\dots, \boldv_r\}\subseteq V\text{,}\) let \(S'=T(S)=\{[\boldv_1]_B,[\boldv_2]_B,\dots, [\boldv_r]_B\}\subseteq \R^n\text{.}\) We have the following equivalences:
  1. \(\boldv\in\Span S\) if and only if \([\boldv]_{B}\in \Span S'\text{;}\)
  2. \(S\) is linearly independent if and only if \(S'\) is linearly independent.
