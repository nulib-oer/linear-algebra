Corollary 3.4.16. Solutions to matrix equations.

Fix an \(m\times n\) matrix \(A\) and column vector \(\boldb\in \R^m\text{.}\) If \(\boldx_p\) is a solution to
\begin{equation} A\boldx=\boldb\text{,}\tag{3.4.4} \end{equation}
then the set \(X_\boldb\) of all solutions to (3.4.4) is described as
\begin{equation*} X=\{\boldx_p+\boldu\colon \boldu\in \NS A\}\text{.} \end{equation*}
In other words, \(X_\boldb\) is the translate of \(\NS A\) by the vector \(\boldx_p\text{.}\)
