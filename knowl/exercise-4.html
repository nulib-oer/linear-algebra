Exercise 2.

For each of the following functions, \(f:\textbf{R}\longrightarrow\textbf{R}\) state whether they are one-to-one, onto, both or neither. Use the following key:
  1. \(\displaystyle f(x)=e^x\)
  2. \(\displaystyle f(x)=x^3+6\)
  3. \(\displaystyle f(x)=x(x-6)(x+6)\)
  4. \(\displaystyle f(x)=\left|x\right|\)
  5. \(\displaystyle f(x)=\sin(x)\)
  6. \(\displaystyle f(x)=x^2+4\)
  7. \(\displaystyle f(x)=x + 4\left|x\right|\)
