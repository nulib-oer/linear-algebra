Consider \(F(\R,\R)\text{.}\) A vector of \(F(\R,\R)\) is a function \(f\colon \R\rightarrow \R\text{:}\) a rule that assigns to any input \(x\in \R\) a unique output \(y\in \R\text{.}\) Thus the functions \(f\) and \(g\) defined as \(f(x)=x^2+1\) and \(g(x)=\sin x-x\) are both vectors of \(F(\R,\R)\text{,}\) as is any function given by a formula involving familiar mathematical functions and operations (as long as the formula is defined for all \(x\in \R\)). Thatâ€™s a lot of vectors! And yet we are only beginning to scratch the surface, since a function of \(F(\R,\R)\) need not be given by a nice formula; it simply has to be a well-defined rule. For example, the function \(h\) defined as
\begin{equation*} h(x)=\begin{cases} 1\amp \text{if } x \text{ is rational}\\ 0\amp \text{if } x \text{ is not rational} \end{cases} \end{equation*}
is also an element of \(F(\R,\R)\text{.}\)
