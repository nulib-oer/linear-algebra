Theorem 3.3.22. Polynomial equality.

Let \(X\subseteq \R\) be a nondegenerate interval, and let
\begin{align*} f(x) \amp =\anpoly \amp g(x)\amp=\bnpoly \end{align*}
be polynomials on \(X\text{.}\)
  1. We have \(f=g\) if and only if \(a_i=b_i\) for all \(1\leq i\leq n\text{.}\)
  2. In particular, \(f=\boldzero\) if and only if \(a_i=0\) for all \(1\leq i\leq n\text{.}\)
