Fix an angle \(\alpha\) and define
\begin{equation*} \rho_\alpha\colon \R^2\rightarrow \R^2 \end{equation*}
to be the function that takes an input \(P=(x,y)\text{,}\) considered as a point in the \(xy\)-plane, and returns the output \(P'=(x',y')\) obtained rotating \(P\) by an angle of \(\alpha\) about the origin. The function \(\rho_\alpha\) is called rotation about the origin by the angle \(\alpha\text{.}\)
