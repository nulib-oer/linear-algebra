Procedure 3.5.12. Investigating linear independence.

  1. Write out the general vector equation
    \begin{equation*} c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n=\boldzero \end{equation*}
    where the \(\boldv_i\) are arbitrary elements of the given set \(S\text{.}\)
  2. Translate this vector equation into a homogeneous linear system in the unknowns \(c_1,c_2,\dots, c_n \text{,}\) using the definition of equality for your vector space.
  3. Decide, using Gaussian elimination, whether this system has any nonzero (i.e., nontrivial) solutions.
