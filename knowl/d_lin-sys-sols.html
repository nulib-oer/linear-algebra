Definition 2.1.5. Solutions to linear systems.

A solution to a linear equation

\begin{equation*} a_1x_1+a_2x_2+\cdots a_nx_n=b \end{equation*}

is an ordered sequence \((s_1,s_2,\dots, s_n)\) of real numbers for which the variable assignment \(x_1=s_1, x_2=s_2,\dots, x_n=s_n\) makes the equation true. We say \((s_1,\dots ,s_n)\) solves the equation in this case.

A solution to a system of linear equations

\begin{equation*} \eqsys \end{equation*}

is a sequence \((s_1,s_2,\dots,s_n)\) that is a solution to each of the system's \(m\) equations. We say \((s_1,s_2,\dots, s_n)\) solves the system in this case.
