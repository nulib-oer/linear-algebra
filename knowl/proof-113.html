Proof

According to TheoremÂ 5.3.28 the matrix \(A\) is similar to a diagonal matrix \(D\) if and only if there is a linear transformation \(T\colon \R^n\rightarrow \R^n\) and ordered bases \(B, B'\) of \(\R^n\) such that \([T]_B=A\) and \([T]_{B'}=D\text{.}\) By definition such a \(T\) would be diagonalizable, since \([T]_{B'}=D\) is diagonal. Since \(T\) is diagonalizable if and only if \(A=[T]_B\) is diagonalizable, we conclude that \(A\) is similar to a diagonal matrix \(D\) if and only if \(A\) is diagonalizable.
