Remark 1.2.9. Negating quantifiers.

Let \(P(x)\) be a propositional function with domain of discourse \(D\text{.}\) According to Definition 1.2.5 a universal quantification \(\forall x\, P(x)\) is false if it is not the case that \(P(d)\) is true for all for all \(d\in D\text{;}\) that is, if there is some \(d\in D\) such that \(P(d)\) is false. Letting \(\neg P(x)\) be the propositional function defined as \(\neg P(d)\) for all \(d\in D\text{,}\) we see that \(\forall x\, P(x) \) is false if and only if \(\exists x\, \neg P(x)\) is true. This proves the equivalence

\begin{equation*} \neg\forall x\, P(x)\iff \exists x\, \neg P(x)\text{.} \end{equation*}

Similarly, we have

\begin{equation*} \neg\exists x\, P(x)\iff \forall x\, \neg P(x)\text{.} \end{equation*}

These equivalences can be understood as rules of negating quantifier statements. They can be summarized as follow: “swap quantifiers and negate the predicate.”
