Item 2.

Square zero matrices \(\boldzero\) are never invertible, since for any square matrix \(B\) of the same dimension we have

\begin{equation*} B\,\boldzero=\boldzero B=\boldzero\ne I\text{.} \end{equation*}

Thus there is no matrix satisfying the inverse property for \(\boldzero\text{.}\)
