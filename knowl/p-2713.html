  1. If \(\ S_1\) and \(\ S_2\) are subspaces of \(R^n\) of the same dimension, then \(S_1=S_2\text{.}\)
  2. If \(\ S =\) span{\(u_1, u_2, u_3\)}, then \(dim(S) = 3\) .
  3. If the set of vectors \(U\) spans a subspace \(S\text{,}\) then vectors can be added to \(U\) to create a basis for \(S\)
  4. If the set of vectors \(U\) is linearly independent in a subspace \(S\) then vectors can be added to \(U\) to create a basis for \(S\)
  5. If the set of vectors \(U\) is linearly independent in a subspace \(S\) then vectors can be removed from \(U\) to create a basis for \(S\text{.}\)
