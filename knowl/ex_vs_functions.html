Definition 3.1.8. Real-valued functions.

Underlying set.
Let \(X\) be a set. The vector space of functions from \(X\) to \(\R\), denoted \(F(X,\R)\text{,}\) is the set of all real-valued functions \(f\colon X\rightarrow \R\text{:}\) i.e., the set of all functions with domain \(X\) and codomain \(\R\text{.}\)
Vector operations.
The vector operations on \(F(X,\R)\) defined below are generalizations of operations you may have seen before when learning about function transformations.
Scalar multiplication. Given \(c\in \R\) and a real-valued function \(f\colon X\rightarrow \R\text{,}\) we let \(cf\) be the function defined as
\begin{equation*} (cf)(x)=c(f(x)) \text{ for all } x\in X\text{.} \end{equation*}
Vector addition. Given real-valued functions \(f\) and \(g\) with domain \(X\text{,}\) we let \(f+g\) be the function defined as
\begin{equation*} (f+g)(x)=f(x)+g(x) \text{ for all } x\in X\text{.} \end{equation*}
Zero vector and vector inverses.
The zero vector \(F(X,\R)\) is the constant function \(O_X\) that assigns a value of 0 to all elements of \(X\text{:}\) i.e., \(0_X(x)=0\) for all \(x\in X\text{.}\)
Given a function \(f\in F(X,\R)\text{,}\) its vector inverse is the function \(-f\) defined as
\begin{equation*} (-f)(x)=-f(x) \text{ for all } x\in X\text{.} \end{equation*}
