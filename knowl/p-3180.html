Let \(B=(\boldv_1,\boldv_2,\dots, \boldv_n)\) be an ordered basis for the vector space \(V\text{.}\)
  1. The function \(T\colon V\rightarrow \R^n\) defined as \(T(\boldv)=[\boldv]_B\) is a linear transformation. We call \(T\) the coordinate vector transformation with respect to \(B\text{.}\)
  2. We have \([\boldv]_B=[\boldw]_B\) if and only if \(\boldv=\boldw\text{:}\) i.e., \(T\) is injective.
  3. For all \(\boldx=(c_1,c_2,\dots, c_n)\in \R^n\text{,}\) there is \(\boldv\in V\) with \([\boldv]_B=\boldx\) : i.e., \(T\) is surjective.
  4. Given a subset \(S=\{\boldv_1,\boldv_2,\dots, \boldv_r\}\subseteq V\text{,}\) let \(S'=T(S)=\{[\boldv_1]_B,[\boldv_2]_B,\dots, [\boldv_r]_B\}\subseteq \R^n\text{.}\) We have the following equivalences:
    1. \(\boldv\in\Span S\) if and only if \([\boldv]_{B}\in \Span S'\text{;}\)
    2. \(S\) is linearly independent if and only if \(S'\) is linearly independent.
