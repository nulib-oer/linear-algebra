Theorem 5.2.9. Computing with matrix representations.

Let \(T\colon V\rightarrow W\) be linear, and let \(B\) and \(B'\) be ordered bases of \(V\) and \(W\text{,}\) respectively. The matrix \(A=[T]_{B}^{B'}\) satisfies the following properties:
  1. \(\boldv\in \NS T\) if and only if \([\boldv]_B\in \NS A\text{;}\)
  2. \(\boldw\in \im T\) if and only if \([\boldw]_{B'}\in \CS A\text{;}\)
  3. \(\{\boldv_1,\boldv_2,\dots, \boldv_r\}\) is a basis of \(\NS T\) if and only if \(\{[\boldv_1]_B, [\boldv_2]_B, \dots, [\boldv_r]_B\}\) is a basis of \(\NS A\text{;}\)
  4. \(\{\boldw_1,\boldw_2,\dots, \boldw_s\}\) is a basis of \(\im T\) if and only if \(\{[\boldw_1]_{B'}, [\boldw_2]_{B'}, \dots, [\boldv_s]_{B'}\}\) is a basis of \(\CS A\text{;}\)
  5. \(\nullity T=\nullity A\) and \(\rank T=\rank A\text{.}\)

Proof.
